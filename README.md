# qdaily

仿好奇心日报app

## 功能列表

- [x] 新闻浏览
- [x] 新闻图片浏览
- [x] 侧边栏
- [x] 新闻分类
- [x] 分享到微信,微博,QQ等社交平台
- [x] 热更新&&自动更新
- [x] 友盟统计
- [ ] 登录注册
- [ ] 评论和点赞
- [ ] LABS细节展示
